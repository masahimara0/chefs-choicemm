# Assets Folder

Bu klasöre aşağıdaki görselleri eklemeniz gerekiyor:

## Gerekli Dosyalar

1. **icon.png** (1024x1024 px)
   - Ana uygulama ikonu
   - Şeffaf arka plan olmadan

2. **splash.png** (1284x2778 px)
   - Açılış ekranı görseli
   - Arka plan rengi: #0f0c29

3. **adaptive-icon.png** (1024x1024 px)
   - Android adaptive icon
   - Sadece ön plan görseli

4. **favicon.png** (48x48 px)
   - Web için favicon

## Hızlı Çözüm

Eğer hızlıca test etmek istiyorsanız, Expo'nun varsayılan ikonlarını kullanabilirsiniz:

```bash
# Expo varsayılan assets'lerini kopyala
npx expo customize
```
